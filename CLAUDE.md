# AI 超级机柜产业链行情看板

## 项目概述
A 股 AI 超级机柜产业链个股行情追踪看板，按"材料层→部件层→设备层"三层架构展示各板块涨跌。

## 目录结构
```
ai_chain/
├── config.json          # 板块与个股配置（编辑此文件增删组/个股）
├── data.json            # 拉取后的行情数据（自动生成）
├── dashboard.html       # 仪表盘前端（单文件，纯静态）
├── fetch_data.py        # 数据拉取脚本（腾讯行情 API）
├── serve.py             # 本地 HTTP 服务器
├── rebuild.py           # 一键重建脚本
└── CLAUDE.md            # 本文件
```

## 数据流
```
config.json → fetch_data.py → data.json → dashboard.html 读取渲染
```

## 常用操作

### 拉取最新数据
```bash
python3 fetch_data.py
```

### 本地预览
```bash
python3 serve.py
# 浏览器打开 http://localhost:8080/dashboard.html
```

### 增删个股
编辑 `config.json` 的 `groups` 数组，在对应组的 `stocks` 中增删即可。

### 新增板块
在 `config.json` 中新增 group 对象：
```json
{
  "name": "1.1 芯片制造材料",
  "layer": "materials",
  "desc": "描述",
  "color": "#颜色",
  "stocks": [...]
}
```
`layer` 取值：`materials` / `components` / `equipment`

## 技术细节
- 前端单文件，无依赖，纯原生 JS
- 数据来源：腾讯行情 API（免费，无需 key）
- 涨跌计算：流通市值加权平均
- 刷新间隔：5 分钟（dashboard.html 中 `REFRESH_INTERVAL` 常量）
- 层级分组：dashboard.html 中 `LAYER_ORDER` / `LAYER_LABELS` 常量控制显示顺序和名称
